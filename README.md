# SDL2 Raytracer

Simple raytracer using SDL2

## Dependencies
* Visual Studio 2022
* [SDL2](https://www.libsdl.org/download-2.0.php)
* [glm](https://github.com/g-truc/glm)

## Features
* Primary rays
* Shadow rays
* Reflection via recursive raytracing
* Phong Illumination
* Cornel Box Model
* Camera movement (slow)
